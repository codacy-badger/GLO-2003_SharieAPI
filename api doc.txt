post listing (json)


Quand nous voulons rajouter une offre, il faut appeler
à l'url /listing via un post. Les données du post doivent
être envoyées sous forme d'objet Json.

Un exemple de POST:
{ id : String
    , title : String
    , owner : Owner
    , description : String
    , email : String
    , availabilities : List Time.Posix
    }
Où Owner est:
{ name : String, 
	phoneNumber : String, 
	email : String 
	}
