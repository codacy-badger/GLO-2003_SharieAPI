Quand nous voulons rajouter une offre, il faut appeler
à l'url /listing via un post. Les données du post doivent
être envoyées sous forme d'objet Json.

Un exemple de POST:
{ id : String
    , title : String
    , owner : Owner
    , description : String
    , email : String
    , availabilities : List Time.Posix
    }
Où Owner est:
{ name : String, 
	phoneNumber : String, 
	email : String 
	}
Aussi, lorsqu'un /listing ce fait, l'API retourne l'header
201 CREATED, avec /listing/<id>. L'id va nous permettre de
retrouver la demande.
En cas d'échec, l'header 500 est retourné.
